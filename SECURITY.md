# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it via email to:

**[security@ainfera.ai](mailto:security@ainfera.ai)**

Please include:

- A description of the vulnerability
- Steps to reproduce (if possible)
- Potential impact assessment
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment:** Within 48 hours
- **Initial Assessment:** Within 1 week
- **Resolution Target:** Depends on severity (see [CVSS](https://www.first.org/cvss/v3.1/specification-document))

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest release | :white_check_mark: |

## Security Practices

- We follow secure coding practices and perform regular code reviews.
- Dependencies are monitored for known vulnerabilities.
- All security-related discussions are handled confidentially.
